# Log Aggregator

Centralized log aggregation tool.

Alat agregasi log terpusat.


## Description

Collects, normalizes, and analyzes logs from multiple sources for security event detection.

## Deskripsi

Mengumpulkan, menormalkan, dan menganalisis log dari berbagai sumber untuk deteksi event keamanan.


## Installation

- Navigate to the project folder
- No dependencies required

## Instalasi

- Masuk ke folder proyek
- Tidak perlu dependensi


## Usage

- Aggregate: python log_aggregator.py /var/log/auth.log /var/log/syslog
- Demo: python log_aggregator.py --demo

## Penggunaan

- Agregasi: python log_aggregator.py /var/log/auth.log /var/log/syslog
- Demo: python log_aggregator.py --demo
